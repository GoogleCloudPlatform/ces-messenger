TC-INIT-003: Error Handling: Missing deployment-id

Test Case: Verify that the component fails gracefully when the required deployment-id attribute is missing.

Steps:

  1. Include the ces-messenger.js script.
  2. Add <ces-messenger> to the body without the deployment-id attribute.

Expected Result: The component should fail gracefully. If show-error-messages is true, an informative error about the missing required attribute should be displayed in the chat window. No chat bubble should appear.
