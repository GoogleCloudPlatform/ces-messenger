ATTR-AUDIO-003: audio-input-mode: SPACE_BAR_TO_TALK

Test Case: Verify that audio is sent only while the spacebar is pressed when audio-input-mode is set to SPACE_BAR_TO_TALK.

Steps:

  1. Set audio-input-mode="SPACE_BAR_TO_TALK".
  2. Open chat.

Expected Result: Audio is only sent while the spacebar is pressed and the chat widget is not in focus.
