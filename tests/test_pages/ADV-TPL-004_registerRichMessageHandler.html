ADV-TPL-004: registerRichMessageHandler()

Test Case: Verify that a custom handler for a rich message can be registered and executed.

Steps:

  1. Register a custom handler for the template ID using registerRichMessageHandler.
  2. Click on an element within the rendered template.

Expected Result: The custom handler function should be executed. In this case, the custom content will be removed from teh page, and the element clicked will be shown in the console logs.
