TC-INIT-004: Component Lifecycle: connectedCallback & disconnectedCallback

Test Case: Verify that the component initializes and cleans up correctly when dynamically added to and removed from the DOM.

Steps:

  1. Click the "Add Component" button to dynamically add the <ces-messenger> element to the DOM.
  2. Observe the component.
  3. Click the "Remove Component" button to dynamically remove the component from the DOM.

Expected Result:
1. The component should initialize and render correctly when added.
2. The component should clean up its resources (e.g., event listeners, timers) without errors when removed.
