ADV-DOM-001: createDomHintTracker() and update()

Test Case: Verify that DOM hints can be created and updated, and are sent to the agent.

Steps:

  1. Create a tracker: cesm.createDomHintTracker({ varName: 'cart_contents', selector: '#shopping-cart' }).
  2. Call tracker.update().
  3. Send a message to the agent.

Expected Result: The agent should receive a query parameter named cart_contents containing the innerHTML of the #shopping-cart element.
